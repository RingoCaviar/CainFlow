# CainFlow 

**CainFlow** 是一款受 ComfyUI 启发的轻量级节点式 AI 编排工具。基于原生网页技术，免去复杂依赖，多端适配且交互极速。

## ✨ 核心特性

- 🚀 **并发分支执行**：全新「就绪即出发」并行引擎。支持异步多线程处理，互不干扰的任务流会自动并行，实时反馈，告别串行排队。
- 🛡️ **极致运行稳定性**：
    - **无感心跳监测**：服务端实时感知前端连接状态。若浏览器意外断开或刷新，系统会自动终止耗时的后端 API 请求。
    - **超长任务扩容**：移除所有死板的超时限制。无论是复杂的高分生图还是超长文本对话，引擎都会耐心处理。
- 🎨 **专业级嵌入式图绘**：内置高性能图片编辑器。支持自由画笔、直线、箭头、形状绘制；配备 HSB 动态色环、20 步撤回记录。
- 📂 **全量工作流管理**：内置侧边栏工作流管理器。支持保存、重命名、批量管理；支持启动自动加载「Default」工作流。
- 🧭 **智能坐标与导航**：
    - **世界原点系统**：引入直观的无限长坐标轴十字线与原点标记。
    - **一键聚焦 (F)**：自动计算选中节点的 Bounding-box 并智能缩放适配视口。
- 📱 **深度响应式布局**：全新的「纵向解耦」架构。在任何极端比例下，工具栏会自动折行，内容区域自动顺延。
- 🔈 **后台通知增强**：
    - **音频预热技术**：通过 Audio Warm-up 机制，彻底解决了浏览器在窗口最小化或处于后台时无法播放任务结束提示音的问题。
- 📝 **智能节点布局**：文本输入框支持内容驱动的自动高度调整；对话与文本输出区域支持滚动浏览，节点宽度受控不会因长文本而无限膨胀。
- ♾️ **120FPS 丝滑画布**：引入「全局交互加速」引擎，即使节点大量存在时也能保持流畅体验。
- 💻 **视觉化控制台**：
    - **ASCII 启动旗帜**：个性化的终端启动界面，并提供便捷的 Ctrl+Click 跳转提示。

## 🚀 快速启动

1. 确保电脑已安装 `Python 3`。
2. 双击运行目录下的 **`start_cainflow.bat`**。
3. 如果未报错，浏览器会自动打开 `http://127.0.0.1:8767`。

## 🔧 最近更新 (v4.7)

### v4.7 (2026-04-03)

**✨ 核心新增**
- **全量节点计时系统 (Node Timing Pro)**：支持实时跳秒反馈、执行耗时持久化及工作流耗时汇总。
- **智能图片保存系统 (Auto-Save Pro)**：支持全局路径记忆，并利用 User Gesture 自动申请目录写入权限。
- **专业级缓存审计引擎**：支持实时审计并专项清理生成历史库、预览资产及本地基础存储数据。
- **架构深度重构**：执行引擎转为 **策略模式 (NodeHandlers)**；实现 **UI 面板管理器 (Panel Orchestrator)** 确保多面板互斥开启。

**🛡️ 安全与性能增强**
- **加固级代理安全 (SSRF Protection)**：后端代理升级“解析后验证”机制，有效屏蔽对本级机或内网私有段的请求。
- **高性能存储审计 (Size Cache)**：引入变量追踪机制，打开面板时无需全量扫描 IndexedDB。
- **导入体验优化**：自动检测脱敏后的 Key 缺失并弹出视觉引导提示。

**🎨 UI/UX 优化与深度修复**
- **资产同步销毁**：手动删除节点时同步抹除关联的图片二进制资产。
- **交互逻辑闭环**：对话布局重构为「紧致模式」；修复选中特效 Bug；更换高对比度蓝勾多选标识；锁定设置面板高度。

### v4.5 (2026-04-02)

**✨ 核心新增**
- **全局网络代理系统**：
    - 支持手动配置 HTTP 代理 IP 与端口。
    - 内置「代理连通性测试」功能，一键验证是否可顺利访问 Google/API 接口。
- **运行前智能校验 (Pre-flight Check)**：
    - 点击「开始运行」时，系统会自动扫描画布节点，若检测到绑定的供应商未配置 API 密钥或智能对话节点内容为空，将弹出中断警告。
