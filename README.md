# CainFlow 

## 🔄 最近更新 (2026-03-27 17:47)
- ⚡ **极致拖拽性能 (v2.3.0)**：针对高负载节点（如保存图片、智能对话）进行了深度重构。
    - **零布局抖动 (Zero Reflow)**：引入端口偏移量预缓存机制，在拖拽期间完全禁用 `getBoundingClientRect` 计算，大幅降低 CPU 开销。
    - **低功耗渲染模式**：拖拽期间连线自动切换为“直线+虚线”模式，松开即还原高质量 Bezier 曲线。
    - **重载组件动态卸载**：拖拽期间自动隐藏高分辨率预览图与富文本区域，禁用毛玻璃滤镜，确保即使在低配设备上也能实现 120 FPS 级丝滑移动。
- 🛠️ **修复与增强**：
    - 修复了被禁用节点重启用时残留的“发光”与“运行中”视觉状态。
    - 优化了图片分辨率自动缩放逻辑，防止超大图引起的 API 崩溃。
    - 增强了通知系统的稳定性，工作流完成时支持系统级 Push 提醒。

**CainFlow** 是一款受 ComfyUI 启发的轻量级节点式 AI 编排工具。基于原生网页技术，免去复杂依赖，双击即得。

## ✨ 核心特性

- ♾️ **极速画布交互**：支持无限平移缩放、右键菜单、以及拖拽/点击双重连线流。
- ⚙️ **「供应商-模型」解耦架构**：全新的 API 配置方案。将 API 地址、密钥（供应商）与具体模型 ID（模型）解耦，支持为同一供应商配置多个模型并设置别名。
- 🕒 **图像历史记录系统**：内置基于 IndexedDB 的持久化画廊，自动记录每一次生成的图片、Prompt 及模型信息。
- 🔍 **双交互模式预览器**：支持 **鼠标滚轮缩放 (10% - 2000%)** 与 **左键平移拖拽**，交互体验完美复刻 ComfyUI，轻松检查生成细节。
- ⚡ **极致 UX 优化**：引入 Ghost Frame 虚拟缩放与高负载渲染拦截；支持 **Ctrl+鼠标框选/多选与批量拖动**；支持同步切换多个选中节点的状态。
- 🛡️ **节点控制**：支持 **一键启用/禁用 (Bypass) 节点**，被禁用的节点在工作流执行时会自动跳过。
- 📝 **全状态可视化**：支持节点 **实时执行计时 (0.1s 精度)**、失败警告弹窗与全局执行日志追踪。点击日志条目可查看完整执行上下文与错误堆栈。

## 🧩 核心节点

- **AI 节点组**: 图片生成 (支持参考与宽高比调节) 、多模态智能对话。
- **功能节点组**: 支持拖拽的图片与文本输入板、内置增强型预览器、本地持久化保存节点。

## 💡 API 配置推荐

为了获得最佳的画图与对话体验，推荐从以下渠道获取 API 密钥：
- **推荐地址**：[6789api](https://www.6789api.top/)
- **使用方法**：项目默认已配置该供应商（GXP），你只需进入设置面板，将购买到的 **API Key** 粘贴到对应位置即可。默认提供 `Banana Pro 2` (生图) 与 `对话` (聊天) 两个预设模型。

## 🚀 快速启动

1. 确保电脑已安装 `Python 3`。
2. 双击运行目录下的 **`start_cainflow.bat`**。
3. 浏览器自动打开 `http://127.0.0.1:8767` 即可开始创作！

*🔒 **隐私提示**：你的 API Key、历史记录、画布逻辑及本地存放位置全权交由你本人的浏览器安全系统（LocalStorage & IndexedDB）托管，无任何后台数据收集。*
