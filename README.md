# CainFlow 

**CainFlow** 是一款受 ComfyUI 启发的轻量级节点式 AI 编排工具。基于原生网页技术，免去复杂依赖，多端适配且交互极速。

---

### 📥 [点击此处一键下载本项目 (ZIP 压缩包)](https://github.com/RingoCaviar/CainFlow/archive/refs/heads/main.zip)

---

## ✨ 核心特性

- 🚀 **并发分支执行**：全新「就绪即出发」并行引擎。互不干扰的独立任务流会自动启动，下游节点在输入数据满足后立即执行，实时反馈结果，告别串行排队等待。
- 🎨 **专业级嵌入式图绘 (NEW)**：内置高性能图片编辑器。支持自由画笔、直线、箭头、形状绘制；配备 HSB 动态色环、20 步撤回记录及无限画布级的平移缩放体验。
- ♾️ **120FPS 丝滑画布**：引入「全局交互加速」引擎，在 2048x2048 高清图片节点中也能保持流畅的平移与缩放体验。
- 📐 **自适应图片容器**：图片预览/导入/保存节点采用弹性布局，大分辨率图片不再撑爆节点。图片始终保持原始比例、自适应填充可用空间。
- 🎆 **呼吸式视觉反馈**：正在运行的节点拥有重型多层青色光晕与「脉冲呼吸」动画，并在层级上自动置顶，一眼锁定核心进度。
- ⚙️ **标签式设置系统 (NEW)**：重构的设置面板支持「API 相关」与「常规设置」标签切换。支持自定义供应商、模型别名及可配置的边长等效分辨率阈值。
- 🧩 **智能拖放交互 (NEW)**：支持将图片直接拖拽到现有「图片导入」节点上进行原地更新，或拖拽到空白处自动创建新节点。
- 🛡️ **静默容错逻辑**：开启「自动重试」后，单个节点失败不会阻塞全局进程，系统将静默记录日志并自动重启失败分支。

## 🚀 快速启动

1. 确保电脑已安装 `Python 3`。
2. 双击运行目录下的 **`start_cainflow.bat`**。
3. 如果未报错，浏览器会自动打开 `http://127.0.0.1:8767`。

## 🧩 核心节点

- **AI 生图**: 支持参考图、多种宽高比与分辨率调节，对接主流推理 API，支持分支并行生成。
- **智能对话**: 多模态 LLM 驱动，流式推断与长文本精细显示。
- **图片工具**: 导入/预览/保存全套方案，支持一键保存到本地目录。支持基于边长的等效分辨率自动缩放。
- **绘图编辑**: 在任何图片预览节点点击「绘画」图标即可进入编辑模式，支持对生成结果进行即时二次创作。

## 🔧 最近更新 (v2.8.0)

- **图片编辑器全量上线**：集成一套完整的 canvas 绘图系统，支持 5 种基础工具及自定义 HSB 调色。内置撤回系统、自动居中适配及坐标同步算法。
- **设置面板标签化**：优化设置面板布局，支持分类管理 API 与通用偏好，提升了在高并发配置下的可用性。
- **智能分辨率管理**：允许用户设置「边长等效分辨率」阈值，当导入超大图片时（如 4K/8K）会自动等比缩小以保护显存。
- **拖拽体验升级**：修复了在节点上释放文件会多创建节点的 Bug，现在支持精确的「覆盖更新」逻辑。
- **节点视觉重构**：为运行中的节点添加了平滑的呼吸灯动画，并将发光半径优化为原来的 50%，保持视觉冲击力的同时减少遮挡。
