# CainFlow

**CainFlow** 是一款受 ComfyUI 启发的轻量级节点式 AI 编排工具。基于原生网页技术构建，开箱即用、依赖简单、交互流畅，适合快速搭建与运行节点式 AI 工作流。

## ✨ 项目亮点

CainFlow 不只是一个简单的 AI 前端，而是一套强调**速度、稳定性、交互体验与本地可控性**的节点式工作流工具。

### ⚡ 性能与执行效率
- **并行执行引擎**：互不干扰的任务分支可自动并行处理，显著提升复杂工作流运行效率。
- **高流畅度画布体验**：通过 DOM 缓存与渲染优化，即使面对大型工作流也依然保持顺滑操作反馈。

### 🛡️ 稳定性与安全性
- **智能心跳监测**：页面关闭或刷新后，服务端可自动清理遗留请求，减少资源浪费。
- **运行前完整校验**：执行前自动检查 API Key 与输入项，降低因配置缺失导致的报错概率。
- **SSRF 防护代理**：内置后端安全代理，适合分布式网络与内网环境使用。

### 🎨 交互与编辑体验
- **无限画布 + 一键聚焦**：面对复杂工作流也能快速定位内容，按下 `F` 可立即回正视图。
- **智能弹性节点布局**：节点支持内容感知尺寸调整，在报错、长文本等场景下可自动扩展。
- **内置图片编辑器**：支持 20 步撤回、HSB 色环与形状绘制，无需依赖第三方工具。
- **响应式多行工具栏**：窗口缩小时按钮可自动换行，工具区与抽屉区域始终保持协调。
- **快捷剪断连接线**：按住 `Ctrl` 并右键划过连线即可快速断开连接。
- **多级撤回 / 重做**：支持画布级 Undo / Redo，降低误操作影响。
- **内置帮助中心**：左侧边栏集成操作说明与快捷键提示，便于快速上手。
- **强制刷新提醒**：内置版本感知提示，避免旧前端缓存影响使用。

### 📦 资产与工作流管理
- **智能重试系统（Auto-Retry+）**：支持自定义最大重试次数，适合高并发或网络波动场景。
- **全量节点计时**：可记录每个步骤耗时，帮助定位性能瓶颈。
- **增强剪贴板系统**：支持图片、文本与内部节点复制粘贴的优先级判断，并具备持久化能力。
- **工作流快照管理**：支持保存、读取与批量重命名，更适合长期积累与整理项目。

## 🆕 v2.7.3 更新内容

本次 **v2.7.3** 重点优化了 OpenAI 兼容生图模型的请求逻辑、参考图上传方式与分辨率配置体验，同时补充了项目架构指引。

### 本次更新重点
- **OpenAI 兼容生图请求自动分流**：无参考图时继续使用 `/v1/images/generations`，当生图节点接入参考图时自动切换到 `/v1/images/edits`。
- **参考图上传格式修复**：OpenAI 兼容图片编辑请求改为 `multipart/form-data`，参考图会作为文件字段上传，解决部分供应商返回 `failed to parse multipart form` 的问题。
- **OpenAI 分辨率菜单优化**：OpenAI 兼容生图模型会显示独立分辨率选项，支持“自动”、`1024×1024`、`2048×2048` 和“自定义”。
- **自定义分辨率输入**：选择“自定义”后可通过“宽度输入框 × 高度输入框”填写尺寸，执行时自动拼成 OpenAI `size` 参数；选择“自动”时不发送分辨率参数。
- **自定义分辨率实时校验**：输入宽高时会即时检查 OpenAI 规范，确保最大边不超过 `3840px`、两边均为 `16px` 倍数、长短边比例不超过 `3:1`，且总像素数位于 `655,360` 到 `8,294,400` 之间；不符合规范时会在输入框下方提示并阻止执行。
- **OpenAI 参数提示与界面简化**：仅在 OpenAI 兼容模型下显示 `Size` 参数兼容性提示，提醒部分中转 API 可能不支持分辨率参数；OpenAI 模型下会隐藏“宽高比”选项，避免展示无效配置。
- **节点数据保存增强**：自定义分辨率会随工作流保存、恢复与节点复制同步保留。

### 延续近期成果
- 继续收敛生图请求、模型协议与分辨率预设到执行模块，减少 UI 与请求逻辑重复维护。
- 更新了 `cainflow-project` Skill 与架构速查表，明确 OpenAI 兼容生图、参考图上传、分辨率菜单和序列化字段的代码落点。
- 保持 Gemini / Google 生图原有宽高比与分辨率配置逻辑不变，避免影响既有工作流。

## 🚀 使用方式

本项目提供两种运行方式，推荐优先使用 **方式一**。

### 方式一：下载发布版（推荐，无需 Python）
1. 前往 [Releases](https://github.com/RingoCaviar/CainFlow/releases/latest) 页面下载最新 `.zip`。
2. 解压文件。
3. 双击目录中的 **`CainFlow.exe`** 直接启动。

> 该版本已内置完整运行环境，无需额外安装 Python。

### 方式二：下载源码运行（适合开发者，需 Python）
1. 下载源码 ZIP，或通过 Git 克隆本项目。
2. 确保本机已安装 **Python 3.x**。
3. 进入项目目录后，双击运行 **`start_cainflow.bat`**。

> 脚本会自动检查依赖并启动本地服务。

> [!NOTE]
> 程序启动后会自动打开 `http://127.0.0.1:8767`。如果浏览器未自动弹出，请手动访问该地址。
