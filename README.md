## ✨ 项目亮点

CainFlow 是一款强调 **速度、稳定性、交互体验与本地可控性** 的轻量级节点式 AI 工作流工具。

### ⚡ 性能与执行效率
- **并行执行引擎**：互不干扰的任务分支可自动并行处理，显著提升复杂工作流运行效率。
- **高流畅度画布体验**：通过 DOM 缓存与渲染优化，即使面对大型工作流也依然保持顺滑操作反馈。
- **灵活节点编排**：支持文本处理、图片生成、视角控制、预览与保存等常用工作流节点。

### 🛡️ 稳定性与安全性
- **智能心跳监测**：页面关闭或刷新后，服务端可自动清理遗留请求，减少资源浪费。
- **运行前完整校验**：执行前自动检查 API Key 与输入项，降低因配置缺失导致的报错概率。
- **SSRF 防护代理**：内置后端安全代理，适合分布式网络与内网环境使用。

### 🎨 交互与编辑体验
- **无限画布 + 一键聚焦**：面对复杂工作流也能快速定位内容，按下 `F` 可立即回正视图。
- **智能弹性节点布局**：节点支持内容感知尺寸调整，在报错、长文本等场景下可自动扩展。
- **内置图片编辑器**：支持 20 步撤回、HSB 色环与形状绘制，无需依赖第三方工具。
- **快捷剪断连接线**：按住 `Ctrl` 并右键划过连线即可快速断开连接。
- **多级撤回 / 重做**：支持画布级 Undo / Redo，降低误操作影响。
- **内置帮助中心**：左侧边栏集成操作说明与快捷键提示，便于快速上手。

### 📦 资产与工作流管理
- **智能重试系统（Auto-Retry+）**：支持自定义最大重试次数，适合高并发或网络波动场景。
- **全量节点计时**：可记录每个步骤耗时，帮助定位性能瓶颈。
- **增强剪贴板系统**：支持图片、文本与内部节点复制粘贴的优先级判断，并具备持久化能力。
- **工作流管理**：支持保存、读取、重命名与预设复用，更适合长期积累与整理项目。

## 🚀 使用方式

本项目提供两种运行方式，推荐优先使用 **方式一**。

### 方式一：下载发布版（推荐，无需 Python）
1. 前往 [Releases](https://github.com/RingoCaviar/CainFlow/releases/latest) 页面下载最新 `.zip`。
2. 解压文件。
3. 双击目录中的 **`CainFlow.exe`** 直接启动。

> 该版本已内置完整运行环境，无需额外安装 Python。

### 方式二：下载源码运行（适合开发者，需 Python）
1. 下载源码 ZIP，或通过 Git 克隆本项目。
2. 确保本机已安装 **Python 3.x**。
3. 进入项目目录后，双击运行 **`start_cainflow.bat`**。

> 脚本会自动检查依赖并启动本地服务。

> [!NOTE]
> 程序启动后会自动打开 `http://127.0.0.1:8767`。如果浏览器未自动弹出，请手动访问该地址。
