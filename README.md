# CainFlow 

**CainFlow** 是一款受 ComfyUI 启发的轻量级节点式 AI 编排工具。基于原生网页技术，免去复杂依赖，多端适配且交互极速。

---

### 📥 [点击此处一键下载本项目 (ZIP 压缩包)](https://github.com/RingoCaviar/CainFlow/archive/refs/heads/main.zip)

---

## ✨ 核心特性

- ♾️ **120FPS 丝滑画布**：引入「全局交互加速」引擎，自动在拖拽/缩放时降级渲染开销，支持 2048x2048 高清图节点流畅交互，交互手感完美复刻。
- ⚙️ **灵活 API 架构**：独创「供应商-模型」解耦设计，支持自定义分发地址与模型别名，内置本地代理转发。
- 🔍 **专业级预览系统**：图片预览/导入/保存节点均支持一键全屏、10%-2000% 缩放与抓取平移，内置 MP 级别自动分辨率优化。
- 🛡️ **稳健运行环境**：启动时自动检测端口占用，支持节点 Bypass、实时执行计时及详细错误堆栈通知。
- 🕒 **高效管理面板**：内置基于 IndexedDB 的持久化画廊，支持多选模式、一键全选及批量下载。

## 🚀 快速启动

1. 确保电脑已安装 `Python 3`。
2. 双击运行目录下的 **`start_cainflow.bat`**。
3. 如果未报错，浏览器会自动打开 `http://127.0.0.1:8767`。

## 🧩 核心节点

- **AI 生图**: 支持参考图、多种宽高比与分辨率调节，对接主流推理 API。
- **智能对话**: 多模态 LLM 支持，流式推断与长文本显示。
- **导入/预览/保存**: 支持 PNG 元数据（即将推出）、导出保存按钮组及高清预览预览。

## 💡 使用提示

- **API 配置**：推荐使用 [6789api](https://www.6789api.top/)。项目默认配置供应商 (GXP)，填入 **API Key** 即可。
- **性能建议**：通过顶部工具栏开启「系统通知」，以便在长任务生成完成后及时收到反馈。
