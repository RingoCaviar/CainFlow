# CainFlow 

**CainFlow** 是一款受 ComfyUI 启发的轻量级节点式 AI 编排工具。基于原生网页技术，免去复杂依赖，多端适配且交互极速。

### 📥 [点击此处一键下载本项目 (ZIP 压缩包)](https://github.com/RingoCaviar/CainFlow/archive/refs/heads/main.zip)

---

## ✨ 核心特性

- 🚀 **并发分支执行**：全新「就绪即出发」并行引擎。互不干扰的独立任务流会自动启动，下游节点在输入数据满足后立即执行，实时反馈结果，告别串行排队等待。
- ♾️ **120FPS 丝滑画布**：引入「全局交互加速」引擎，在 2048x2048 高清图片节点中也能保持流畅的平移与缩放体验。
- 📐 **自适应图片容器**：图片预览/导入/保存节点采用弹性布局，大分辨率图片不再撑爆节点。用户可自由缩放节点大小，图片始终保持原始比例、自适应填充可用空间。
- 🎆 **高能视觉反馈**：正在运行的节点拥有重型多层青色光晕（Halo）与脉冲呼吸效果，并在层级上自动置顶，一眼锁定核心进度。
- ⚙️ **灵活 API 架构**：独创「供应商-模型」解耦设计，支持自定义分发地址与模型别名，内置本地代理转发。
- 🧩 **精修端口渲染**：重构节点阴影与端口布局，所有输入输出小圆点完美内嵌于边框内，彻底解决了在各种浏览器缩放倍率下的裁剪问题。
- 🔍 **专业级预览系统**：图片预览/导入/保存节点均支持一键全屏、10%-2000% 缩放与抓取平移，内置自动分辨率优化。
- 🛡️ **静默容错逻辑**：开启「自动重试」后，单个节点失败不会阻塞全局进程，系统将静默记录日志并自动重启失败分支，不弹出全屏错误框。
- 🕒 **高效管理面板**：内置基于 IndexedDB 的持久化画板状态，支持多选模式、一键全选及批量下载。

## 🚀 快速启动

1. 确保电脑已安装 `Python 3`。
2. 双击运行目录下的 **`start_cainflow.bat`**。
3. 如果未报错，浏览器会自动打开 `http://127.0.0.1:8767`。

## 🧩 核心节点

- **AI 生图**: 支持参考图、多种宽高比与分辨率调节，对接主流推理 API，支持分支并行生成。
- **智能对话**: 多模态 LLM 驱动，流式推断与长文本精细显示。
- **图片工具**: 导入/预览/保存全套方案，支持一键保存到本地目录（支持路径持久化）。节点可自由缩放，大图自适应显示不变形。

## 💡 使用提示

- **节点缩放**：拖拽节点右下角的把手可自由调整节点大小，图片容器会自动适配，无需担心大分辨率图片撑爆布局。
- **并发建议**：多分支并行可能受 API 侧并发限制，建议合理安排任务流。
- **API 配置**：默认供应商 (6789api)，填入 **API Key** 即可享受 0.1 元生成的 Banana Pro 2 系列模型。
- **性能建议**：通过工具栏开启「运行通知」，生成长任务时可切屏操作，完毕后系统会主动推送提醒。

## 🔧 最近更新 (v2.7.0)

- **画布缩放渲染优化**：修复了缩放过程中文字模糊的问题，引入「像素级延迟刷新」逻辑，确保在大倍率缩放后文字依然保持 1:1 的清晰度。
- **智能连线高亮系统**：节点间的连线默认为沉浸式淡灰色且无发光，仅在选中相关节点时瞬时转变为「纯紫色高亮实线」，大幅降低了复杂工作流中的视觉噪音。
- **高性能 3D 转换**：全量启用 `translate3d` 硬件加速，在低功耗设备上平移画布也有极佳的响应性能。
- **修复全屏预览内存泄漏**：每次打开全屏预览不再累积 window 级事件监听器。
